IIFE(Immediatley invoked function expression).

IIFE is used in javascript to avoid the need to create global variables or functions.

This is used in case we have more than one javascript files attached to SAME html.

These files have the sameGLOBAL SCOPE and hence might lead to problems in case of variables and functions with same name.

If you want variables with the same names in both the JS files without having any error, then you can enclose your code within curly braces to make its scope block-level. Then use the let keyword to declare your variables within them.
